# Cross-cohort Tau Harmonization Analysis

Code for the cross-cohort tau-positivity harmonization study across ADNI, OASIS3, and NACC.

## Contents
- severity-strip ablation
- matched-feature rerun
- OASIS feature audit
- bootstrap confidence intervals
- publication table/figure generation

## Important
This repository contains code and manuscript assets only.
It must not contain participant-level ADNI, OASIS3, or NACC data, or derived participant-level tables.

## Main scripts
- `crosscohort_tau_severity_strip.py`
- `crosscohort_tau_matched_rerun.py`
- `audit_oasis_feature_availability.py`
- `bootstrap_crosscohort_auc_ci.py`
- `plot_crosscohort_severity_figure.py`
- `build_table1_and_figure3.py`
- `rebuild_crosscohort_publication_tables.py`
